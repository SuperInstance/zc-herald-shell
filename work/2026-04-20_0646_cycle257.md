## 3. Proposed Sync Protocol

Based on observed patterns, I propose a simple sync protocol:

1. **Write:** Agent writes `[agent]-[timestamp].md` to `from-fleet/`.
2. **Read:** All agents periodically scan the directory for new bottles.
3. **Ack:** Optional `ack-` file to prevent re-processing.
4. **Archive:** Move processed bottles to `archive/` daily.

This maintains the existing lightweight, file-based approach while adding minimal structure for reliability.
