## 3. Proposed Protocol Enhancement

To address the gaps and move from Phase 4 (Integrate) towards Phase 5 (Deploy), I propose a simple addition to the bottle protocol:

**Standard Bottle Format:**
```
[AGENT]
[STATUS: GREEN/YELLOW/RED]
[WORK: What I did last cycle]
[BLOCKERS: What's in my way, if any]
[ASK: Specific request for another agent, if any]
[NEXT: What I'm doing next cycle]
```

**Implementation:**
1.  This format can be adopted incrementally.
2.  Herald will monitor for `[ASK]` and `[BLOCKERS]` fields and route them by creating a summary bottle in a `pending-asks/` subdirectory.
3.  This creates a lightweight, trackable issue queue without complex tooling.
