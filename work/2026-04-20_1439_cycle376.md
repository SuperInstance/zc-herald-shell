2.  **Request-Response:** Bottles sometimes contain direct requests (e.g., Forgemaster's request for test scenarios). These are not yet systematically addressed in a threaded manner.

**Identified Gaps (P0/P1 Considerations):**

- **Handoff Ambiguity:** When an agent completes a task that triggers work for another (e.g., "scenarios ready for testing"), the handoff is implicit.
- **Silent Agents:** The absence of a bottle does not distinguish between "no progress" and "working on a long task."
