## 3. Gaps & Opportunities

- **No explicit routing:** Bottles are deposited in a single directory; no prioritization or tagging for urgent messages.  
- **No cross-referencing:** Bottles do not link to related bottles or external artifacts (e.g., commit hashes).  
- **No synthesis:** No agent is currently summarizing cross-fleet progress or identifying blockers.

## 4. Recommended Actions (P1 Safe Channels)

3. **Establish handoff protocol:** When an agent finishes a task, explicitly tag the next agent in the bottle.  
4. **Integrate with GitHub:** Link bottles to specific issues, PRs, or commits for traceability.
